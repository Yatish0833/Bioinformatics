    Multiple sequence alignment tool

    Multiple sequence alignment tool which uses a progressive algorithm that allows pairwise alignments using ktuple method and then uses that score for guided tree construction

    Advantages:

    1.Most memory efficient

    2.Good results on Large dataset espcially with long N/C terminals

    3.Fast algorithm

    Disadvantages:

    1.Errors are not corrected

    2.Hard to know if solution is good

    Clustalo

    Multiple sequence alignment tool

    Multiple sequence alignment tool which uses an iterative algorithm which involves kimura distance method and UPGMA method to produce guide trees

    Advantages:

    1.No limitations on number of input sequences to be aligned

    2.Fast and accurate

    Disadvantages:

    1.only acceptable format is FASTA
